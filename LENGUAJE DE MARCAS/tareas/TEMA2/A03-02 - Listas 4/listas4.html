Definicion de terminos de desarrollo web

Frontend
La parte visible de una aplicaion web o sitio web con la que el usuario interactua.
Backend
La parte de una aplicacion que maneja la base de datos, servidores y logica del servidor.
API
Interfaz que permite que diferentes apliaciones se comuniquen entre si.
Contrato entre dos apliaciones que se especifica como se intercambian los datos.
Framework
Una estructura de soporte que proporciona herramientas y blibliotecas para desarrollar hardware.
Conjunto de herramientas y blibliotecas preconfiguradas que facilitan el desarrrollo de aplicaciones al proporcionar una estrucutra base sobre la que se puede consturir el codigo.
Conjuntos de relgas y estandares que guina el desaroolo de software dentro de un area especifica
Marco de trabajo o modelo para contruir sistemas o apliacaionbes complejas de manera mas eficiente
