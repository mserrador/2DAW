Programación FrontEnd

Lenguajes básicos del FrontEnd

HTML
Lenguaje de marcado utilizado para estructurar contenido en la web.
CSS
Lenguaje utilizado para dar estilo a los elementos HTML.
JavaScript
Lenguaje de programación que permite la interactividad en las páginas web.

Tecnologías usadas en el FrontEnd

CSS
Lenguaje de hojas de estilo utilizado para definir la apariencia visual de las páginas web.
Permite separar la estructura HTML de la presentación, facilitando el diseño responsivo.
Ofrece propiedades para controlar el color, el tamaño, el espaciado, el diseño de la página y más.
JavaScript
Lenguaje de programación utilizado para crear interactividad en las páginas web.
Permite manipular el contenido del HTML y CSS a través del DOM (Document Object Model).
Es un lenguaje de alto nivel que soporta programación orientada a objetos, funcional y basada en eventos.
Es fundamental para el desarrollo de aplicaciones web dinámicas y modernas.
HTML
Lenguaje de marcado estándar para estructurar contenido web.
