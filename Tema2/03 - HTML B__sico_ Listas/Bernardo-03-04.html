Definición de términos de desarrollo web

Frontend
La parte visible de una aplicación web o sitio web con la que el usuario interactua.
Backend
La parte de una aplicación que maneja la base de datos, servidores y lógica del lado del servidor.
API
Interfaz que permite que diferentes aplicaciones se comuniquen entre si.
Contrato entre dos aplicaciones que especifica cómo se intercambian datos.
Framework
Una estructura de soporte que proporciona herramientas y bibliotecas para desarrollar software.
Conjunto de herramientas y bibliotecas preconfiguradas que facilitan el desarrollo de aplicaciones al proporcionar una estructura base sobre la que se puede construir el código.
Conjunto de reglas y estándares que guían el desarrollo de software dentro de un área específica.
Marco de trabajo o modelo para construir sistemas o aplicaciones complejas de manera más eficiente.
